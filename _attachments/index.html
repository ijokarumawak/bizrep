Info 利用ソフトウェア一覧

  • Apache CouchDB

    RESTfulなAPIを備えたドキュメント指向データベース。Webサーバ機能も持っており、Javascriptで簡単なWebアプリケーションを開発できる。

  • CouchApp

    CouchDBでのWebアプリケーション開発を便利にするツール。アタッチメントやview、list、showなどの各種Javascriptファイルをローカルファイルシステム上で作成し、CouchDBにデプロイする仕組みを提供する。

  • Twitter Bootstrap

    Twitterで開発しているユーザいたフェース用ライブラリ。シンプルなHTMLをベースに、CSSとJavascriptを利用してモダンなユーザインタフェースを作成することができます。

  • jQuery

    HTMLドキュメントの操作、イベントハンドリング、アニメーションなどを行うJavascriptライブラリ。

    jQueryプラグイン一覧

    jQuery hashchange event ブラウザの戻るボタンや、ブックマークに対応するためのプラグイン。
